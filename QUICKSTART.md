# YouTube Upload Tool

## Features
- Desktop GUI application (macOS & Windows)
- CLI tool for automation
- Upload videos with metadata
- Download videos from YouTube
- Manage your channel videos

## Quick Start

### Desktop App
```bash
cd desktop-app
./install.sh  # macOS
install.bat   # Windows
./youtube-uploader
```

### CLI Tool
```bash
cd cli-tool
./youtube --upload video.mp4 --title "My Video"
```
